Convexity

We start the semester with a lecture on convexity. Interesting note that there is always a set of hyperplanes, defined by simple relations on the orginal convex set, that bound the set when taken together.

Conjugate variables are mentioned here, but I didn't catch where they were defined. However, we can make a function from these conjugates, and it effectively is the slope of the original convex function. This is the 'duality' in convex theory that leads so nicely to its properites in optimization.

Sneaky Proof Technique

Proving a = b is done as both a<= b and b <= a. We get one of these for free with convex functions, when consdering the optimal value.

Convex Optimization

We get a basic rundown of the classic optimization problem formulation. If all of the inequality constraints (and objective function) are convex, and the equality constraints affine, this optimization problem is convex, which is a very handy property to have, as it allows us to solve the problem in about the same time complexity as a similarly sized linear programming problem.

We note that the equality constraints being affine (e.g. 'linear' in higher order spaces) comes from needing to be convex both for h(x) = 0 and -h(x) = 0. The only functions for which both of those equalities hold are affine, which in the notes we write henceforth as $h_j(x) = a_j^Tx - b_j$.

Feasibility

I like this definition, its compact, so I want to write it down. The feasible set of an optimization problem is

$D = dom(f_0) \cap {x| f_i(x) \leq 0} \cap {h_j(x) = 0}$

For a convex optimization problem, this set is always convex, but it is not guaranteed to be nonempty. If $D = \emptyset$, then $p = + \infty$.

Optimality

When we have a tangent plane, we can write the optimality condition as $\nabla f_0 = \lambda \hat{n}$. This depends on smoothness, of course, for the existence of the tangent plane. This condition is equivalent to the Karush-Kuhn-Tucker conditions, and combined with $\nabla f_0(x)^T(y-x) \geq 0$ for all $y \in D$, is necessary and sufficient for an optimum solution.
