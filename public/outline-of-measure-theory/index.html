The exterior measure

Loosely speaking, the exterior measure (also called the outer measure or outer content) m* assigns to any subset of Rd a first notion of size. However, the exterior measure lacks the desirable property of additivity when taking the union of disjoint sets. The exterior measure, as the name indicates, attempts to describe the volume of a set E by approximating it from the outside. The set E is covered by cubes, and if the covering gets finer, with fewer cubes overlapping, the volume of E should be close to the sum of the volumes of the cubes. The precise definition is

Example: the Cantor set. From its construction, we know that C is a subset of Ck for each k, and Ck is a disjoint union of 2k closed intervals, each of length  3-k. Consequently, m*(C) <= (2/3)k for all k, and thus m*(C) = 0.

Properties of the exterior measure:

  • Monotonicity: If E1 is a subset of E2, then m*(E1) < m*(E2)
  • Countable sub-additivity: If E is a countable union of subsets Ej, then m*(E) <= sum(m*(Ej))
  • If E is a subset of Rd, then m*(E) = inf m*(O), where the infimum is taken over all open sets O containing E.
  • If E = E1 U E2, and d(E1,E2) > 0, then m*(E) = m*(E1) + m*(E2)
  • If E is the countable union of almost disjoint cubes Qj, then m*(E) = sum(|Qj|)

Measurable sets and the Lebesgue measure

It turns out, that in general we cannot conclude that (E = E1 U E2, E1 intersect E2 = null) -> m*(E) = m*(E1) + m*(E2) for countable unions of sets in Rd. In order for this to be true, we require the Ej to be measurable sets. A subset E of Rd is Lebesgue measurable, or simply measurable, if for any e > 0 there exists an open set O with E a subset of O, and m*(O-E) <= e.

Thus, for a set to be nonmeasurable, we require that for some e > 0, no such open set exists. This seems difficult to find, a priori, since by property 3 of the exterior measure, m*(E) is defined as the exterior measure of the smallest open set containing E, so it seems trivial to find an open set O st m*(O - E) < e. Thinking it through, though, the definition of measurable uses the measure of the set difference, not the difference of the measures of the sets. So maybe this does make sense.

Properties of the Lebesgue measure (inherits 1-5 above from the exterior measure):

  • Every open set in Rd is measurable
  • If m*(E) = 0, then E is measurable. In particular, if F is a subset of a set of exterior measure 0, then F is measurable.
  • A countable union of measurable sets is itself measurable.
  • Closed sets are measurable
  • The complement of a measurable set is measurable.
  • A countable intersection of measurable sets is measurable.

Theorem: If E1, E2, … are disjoint measurable sets, and E is their (countable) union, then m(E) = sum(m(Ej)).

Construction of a non-measurable set

Take x, y in [0,1]. We write x ~ y whenever x - y is rational, and note that this is an equivalence relation. Two equivalence classes are either disjoint or coincide, and [0,1] is the disjoint union of all equivalence classes. Now we construct the set N by choosing exactly one element xa from each equivalence class, and setting N = {xa}. This set is not measurable. The proof relies on forming new sets Nk, which are translates of N by the rationals in [-1,1]. Clearly, [0,1] is a subset of Nk, which is a subset of [-1, 2], and by assumption Nk are measurable since N is (assumed). Then 1 <= sum(m(Nk)) <= 3, and since Nk are translates of N, then 1 <= sum(m(N)) <= 3 as well. This is a contradiction, since neither m(N) = 0 nor any m(N) > 0 satisfy this relationship. N is not measurable.

Measurable functions

The characteristic function of a set is the function which returns one if the input is in the set, and 0 otherwise. A step function is a finite sum f = sum(1,N,ak*chirk), where ak are constants and Rk are rectangles. A simple function is a finite sum f = sum(1,N,ak,chiEk), where ak are constants and Ek are measurable sets.

All of the above sets are measurable. A measurable function is defined as follows: A function f defined on measurable subset E of Rd is measurable, if for all a in R, the set f-1([-inf, a)) = {x in E | f(x) < a} is measurable. The < is not unique or even necessary, and the definition works for preimages of closed and open sets, as well. Stein makes this point via the properties of measurable sets and complements, since {x in E | f(x) >= a} is the complement of {x in E | f(x) < a}, and so if one is measurable, the other must be, since E itself is measurable.

Properties of measurable functions:

  • A function f is measurable if and only if f-1(O) is measurable for every open set O, and f-1(F) is measurable for every closed set F.

  • If f is continuous, then f is measurable. If f is measurable and finite valued, and phi is continuous, then phi(f) is measurable (composition).

  • Suppose fn is a sequence of measurable functions. Then sup fn, inf fn, lim sup fn, and lim inf fn are all measurable.

  • If fn is a collection of measurable functions, and lim n-> inf fn = f, then f is measurable.

  • If f and g are measurable, then

  • The integer powers are measurable (fk, k in Z).

  • f + g and fg are measurable if both f and g are finite valued.

  • Suppose f is measurable, and g = f for almost every x. Then g is measurable.

Theorem 4.1: Suppose f is a non-negative measurable function on Rd. Then there exists an increasing sequence of non-negative simple functions phik that converges pointwise to f.

Theorem 4.2: suppose f is measurable on Rd. Then there exists a sequence of simple function phik such that |phik| <= |phik+1|, and lim k-> inf phik = f for all x.

Theorem 4.3: Suppose f is measurable on Rd. Then there exists a sequence of step functions that converges pointwise to f for almost every x.

Littlewood's three principles

  1. Every set is nearly a finite union of intervals
  2. Every function is nearly continuous
  3. Every convergent sequence is nearly uniformly convergent

The sets and functions above are assumed to be measurable.

Egorov's theorem (4.4): Suppose fk is a sequence of measurable functions defined on a measurable set E, with m(E) < inf, and assume that fk -> f almost everywhere on E. Give e >0, we can find a closed set Ae ⊂ E such that m(E - Ae) <= e and fk -> f uniformly on Ae.

Lusin's theorem (4.5): Suppose f is measurable and finite valued on E with E of finite measure. Then, for every e > 0 there exists a closed set Fe, with Fe ⊂ E, and m(E - Fe) <= e, and such that f|Fe is continuous. (the restriction of f to the set Fe).
